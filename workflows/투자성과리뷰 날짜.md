# 투자 성과 리뷰 일정 확인 가이드 (Review Scheduler)

본 가이드(SOP)는 과거 유망 종목 분석 리포트 중 현재 시점에서 성과 리뷰를 작성해야 하는 일자를 정밀하게 산출하기 위한 절차를 정의합니다.

## 1. 실행 명령 (Trigger)
- 사용자가 "**투자 성과 리뷰 날짜 알려줘**", "**오늘 리뷰할 날짜가 뭐야?**", "**제네시스 일정 확인해줘**"와 같이 요청할 때 실행합니다.

## 2. 분석 데이터 소스
1. **우선 확인:** [Picks Feedback 목록](https://stockanalysis2.pages.dev/picks/feedback) - 기존 리뷰 수행 현황 및 목록을 확인하여 리뷰 중복을 방지하고 우선순위를 파악합니다.
2. **대조 확인:** [Picks 원본 목록](https://stockanalysis2.pages.dev/picks) - 최초 유망 종목이 추천된 날짜와 종목별 성격(Term) 데이터를 수집합니다.

## 3. 리뷰 주기 기준 (Review Interval)
성격별로 다음 주기에 도달했을 때 성과 리뷰를 수행합니다.

| 종목 구분 | 1차 리뷰 시점 | 2차 리뷰 시점 | 비고 |
| :--- | :--- | :--- | :--- |
| **⚡ 단기 유망** | 추천 **7일 후** | - | 최종 수익률 확정 |
| **📈 중기 유망** | 추천 **3개월 후** | 추천 **6개월 후** | 중간 점검 및 최종 성과 |
| **💎 장기 유망** | 추천 **6개월 후** | 추천 **12개월 후** | 정기 성과 점검 |

## 4. 판단 및 보고 절차 (Process)
1. **리스트 매칭:** `feedback` 페이지의 기존 데이터와 `picks` 페이지의 원본 발행일을 대조합니다.
2. **경과일 계산:** 추천일로부터 오늘(`2026-03-28`)까지의 시간을 계산합니다.
3. **대상 선정:** 위 `3항`의 주기에 딱 맞거나, 이미 주기를 지났음에도 리뷰가 작성되지 않은 날짜를 추출합니다.
4. **결과 보고:** "오늘 리뷰를 진행해야 할 분석 일자는 **YYYY-MM-DD** 입니다."라고 명확히 안내합니다.

## 성과 리뷰 필요 일자 자동 산출
1. [Picks Feedback](https://stockanalysis2.pages.dev/picks/feedback) 페이지를 검색하여 기존 리뷰 현황 대조
2. [Picks 원본 페이지](https://stockanalysis2.pages.dev/picks)에서 과거 분석 일자와 종목 성격(단기/중기/장기) 정보 수집
3. 추천일 기준 현재(2026-03-28)가 리뷰 필수 주기(7일/3개월/6개월/12개월)에 해당하는 일자 탐색
4. 리뷰 리스트를 우선순위별로 정리하여 사용자에게 보고
